ABOUT

The Product

Burdoc is a modern markdown editor for the cloud, created with Node.js. It was created as a practice project by the author as he was learning Node.

Technologies Used:

  • Node.js
  • Bootstrap
  • Vue.js
  • Font Awesome
  • PostgreSQL
  • Sequelize
